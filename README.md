# 🎓 Student Placement Prediction System

A Machine Learning based web application that predicts whether a student is likely to get placed based on academic performance and skill-related factors.

The application is built using **Python, Scikit-learn, Pandas, and Streamlit** and is deployed online using **Streamlit Cloud**.

# 📌 Features

- Predicts student placement status
- Simple and interactive user interface
- Real-time prediction system
- Machine Learning based classification model
- Fully deployed web application
- Easy to use and responsive interface

---

# 🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Pickle

---

# 📊 Input Parameters

The prediction is based on the following inputs:

- CGPA
- Number of Internships
- Number of Projects
- Aptitude Test Score

---

# 🤖 Machine Learning Model

The project uses **Logistic Regression** for classification.

### Workflow:
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Prediction
7. Deployment using Streamlit

# ⚙️ Installation & Setup

## Clone the Repository

```bash
git clone https://github.com/your-username/placement_app.git
```

## Navigate to the Project Folder

```bash
cd placement_app
```

## Install Required Libraries

```bash
pip install -r requirements.txt
```

## Run the Streamlit Application

```bash
streamlit run app.py
```
